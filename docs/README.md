## Adding the README screenshot

The main [`README.md`](../README.md) shows a hero image at `docs/screenshot.png`.
It's a placeholder until you add your own:

1. Run the card locally (`node build.js && wrangler dev`) or open your deployed URL.
2. Capture a screenshot (or a short GIF) of your board with a few notes pinned.
3. Save it as `docs/screenshot.png` (or update the image path in `README.md`).

Keep it lightweight (ideally under ~1 MB) so the README loads quickly.
